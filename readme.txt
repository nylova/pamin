Dans ce projet nous avons mis en place les fonctionnements suivants:
 1- mis en place du systeme de gestion d'utilisateur 
 2- systeme de gestion de sante

Nous avons utiliser le bundle :
Security pour la gestion d'utilisateur
FOSRestBundle pour la production de l'api systeme de sante 

Dans la partie systeme de gestion utilisateur,nous avons mis en place :
une interface simple pour le CRUD de l'utilisateur par status.
Un endpoint CRUD pour gérer une liste des docteurs
Un endpoint CRUD pour gérer une liste des patients
Un endpoint CRUD pour gérer une liste des rendez-vous
